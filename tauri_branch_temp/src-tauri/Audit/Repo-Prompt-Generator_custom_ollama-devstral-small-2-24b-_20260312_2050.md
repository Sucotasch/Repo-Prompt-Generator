**Target Repo (main):**
- Веб-приложение на Node.js/Express с фронтендом на React 19
- Использует Vite для сборки
- Основной фокус на RAG с использованием Ollama
- Поддержка Qwen и Gemini через API

**Reference Repo (tauri_proxy):**
- Десктопное приложение на Rust с использованием Tauri
- Фронтенд на React 19
- Более глубокий аудит производительности
- Экспериментальный статус

### 2. Подход к анализу кода

**Target Repo:**
- Использует семантический поиск через эмбеддинги
- Последовательная загрузка файлов из GitHub
- Простая фильтрация файлов

**Reference Repo:**
- Внедрил эвристический скоринг файлов
- Параллельная загрузка файлов
- Гибридный подход (RAG + эвристики)

### 3. Производительность

**Target Repo:**
- Более медленная загрузка файлов (последовательные запросы)
- Слабая приоритизация файлов
- Потенциальные проблемы с памятью

**Reference Repo:**
- Более быстрая загрузка (параллелизм)
- Умная приоритизация файлов
- Лучшее управление ресурсами
